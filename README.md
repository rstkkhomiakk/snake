# Simple Snake Game 🐍
Класична аркадна гра «Змійка», реалізована мовою Python з використанням бібліотеки **Pygame**. Це легкий та функціональний проект, що демонструє роботу з ігровими циклами, обробкою подій та механікою колізій.

## 🚀 Особливості
- **Механіка росту:** Змійка стає довшою після кожного поїдання зеленої їжі.
- **Логіка колізій:** Гра закінчується при зіткненні зі стінами або власним хвостом.
- **Візуалізація рахунку:** Поточний результат відображається у верхній частині екрана під час гри.
- **Інтуїтивне керування:** Керування здійснюється стрілками клавіатури з блокуванням розвороту на 180°.

## 🛠 Технології
- **Python 3.9**
- **Pygame** — для рендерингу графіки та обробки логіки гри.

## 🎮 Як запустити
1. Переконайтеся, що у вас встановлено Python.
2. Встановіть бібліотеку Pygame, якщо вона ще не встановлена:
   `pip install pygame`
